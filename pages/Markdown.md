- Heading
	- `# text` -> heading 1, `## text` -> heading 2
- Bold and italic
	- Bold ⇒ `**text**`
	- Italic ⇒ `*text*`
	- Bold and italic ⇒ Three asterics
- Quote #card
	- 在行首添加>
	- 嵌套引用 ⇒ 每多一级加一个>
	- > This is a test
- List
	- 有序列表：行首添加数字"1." "2." etc.
	- 无序列表：行首添加+或-
